<div class="wikistyle"><h1>JQuery Slide</h1></div>


<h3>Easy way to create slides of images using JQuery.

<h2>Install</h2>
Create a <b>index.html</b>:

<code>
<script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.4.2/jquery.min.js"></script>
</code>	
<code>
<script type="text/javascript" src="slide.js" ></script>
</code>	


<h2>Usage</h2>
Create your slides like that:


	  <div id="slideshow">

	    <div id="slidesContainer">

	      <div class="slide">
	        <h2>Ruby</h2>
	        <p>Ruby Programming</p>
	      </div>

	      <div class="slide">
	        <h2>Java</h2>
	        <p>Java Programming</p>
	      </div>	
	
		 <!-- Another slides here -->

		</div>

	</div>		



Then..just call:


	$("#slideshow").slide(100);
